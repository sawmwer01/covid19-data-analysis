# COVID-19 Data Analysis System
   
   A Python-based analysis system for COVID-19 global data.
   
   ## Data Source
   Our World in Data - COVID-19 Dataset
   
   ## Requirements
   - Python 3.7+
   - pandas
   - matplotlib
   - seaborn
   - plotly
   
   ## Usage
```python
   python main.py
```
